# SOUNDDANCE

**An interactive sound-reactive visual art engine.**

---

## Artist Statement

SoundDance exists at the intersection of sound, light, and human movement. It is built on a simple premise: that music is not just heard — it is seen, felt, and inhabited.

Every visualization in this project is a living translation of audio into form. Bass frequencies become gravitational force. Treble becomes light. Silence becomes negative space. The screen does not play a recording — it listens, and it responds.

This work is designed for the stage. For the LED wall behind the performer. For the moment when the beat drops and the room transforms. It is not a screensaver. It is an instrument — one that turns every microphone-equipped space into a canvas where sound paints itself in real time.

The monochrome modes strip away color to reveal pure structure. The 3D modes create depth where there is only a flat screen. The classic modes honor the long history of audio visualization while pushing it toward something more visceral, more immediate.

SoundDance is not meant to be watched passively. It is meant to be danced in front of. It is meant to make the audience forget where the music ends and the visuals begin.

**Let the sound move. Let the light follow.**

---

## Features

- **21 real-time visualizations** across 4 categories
- **Live microphone input** — reacts to any sound in the room
- **Beat detection** with automatic palette shifting
- **Fully responsive** — scales from phone screens to massive LED walls
- **Zero dependencies** — single HTML file, works offline
- **Fullscreen stage mode** — all UI disappears for clean performance visuals
- **Auto-rotate** — cycles through visualizations hands-free

### Monochrome
Smoke Waves, Ink Splatter, Noise Grid, Light Rays, Matrix Rain, Oscilloscope, Shatter Glass, Pulse Radar, Gravity Well

## Controls

| Key | Action |
|-----|--------|
| `F` | Toggle fullscreen |
| `A` | Toggle auto-rotate |
| `1`-`9`, `0` | Jump to visualization |
| `Arrow Keys` | Cycle prev/next |
| `Tab` | Toggle sidebar |
| `Esc` | Hide sidebar |

## Usage

Open `index.html` in any modern browser. Click **[ INITIALIZE ]**. Allow microphone access. Dance.

For stage use: go fullscreen (`F`), enable auto-rotate (`A`), and let it run. Hover top-left corner to access the sidebar if you need to switch visuals manually.
